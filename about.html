Sobre mim

Bem-vindo ao meu portifólio! Sou um desenvolvedor back-end apaixonado por criar soluções eficientes e escaláveis usando Python. Com sólida expertise em estruturas de dados e algoritmos, minha abordagem técnica busca superar desafios complexos.

Além disso, atuo como analista de dados, utilizando Python para manipulação e análise, Excel para visualizações detalhadas, Power BI para relatórios dinâmicos e SQL para consultas precisas. Minha habilidade em combinar destreza técnica com análise aguçada permite extrair insights valiosos, transformando dados em informações significativas. Estou entusiasmado para enfrentar novos desafios e colaborar em projetos que demandem essa combinação única de habilidades.
